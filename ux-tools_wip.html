UX Resources (W.I.P.)

An ongoing open source project.

April 29th, 2019

Overview

While in school studying UX I realized a problem that alot of students were facing: too many resources. There are too many tools, articles, and books to keep up with. How do you know which ones are worth the time? Well, I decided to UX this problem. I'll be writing daily updates here. Feel free to reach out and join!

April 29th

I'd heard about the Glide App that allows you to turn a Google spreadsheet into an "App", so I'd beek looking for a project to test it on. I was well aware of the problem of resource overload, so I thought "Hey, let's try this!" I made a quick prototype. I started to realize that maybe I should fully ux this project and take some steps back.

June 8th

I reached out to a former classmate and talented researcher, Jocelyn Strong. We discussed the vision and the anecdotal evidence I observed about the problem. We both agreed that we should fully "UX" this project and start with research. We need to validate problem-solution-fit.
